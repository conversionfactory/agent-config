## Task

Debug: $ARGUMENTS

Follow this systematic approach:

1. **Reproduce**: Confirm how to trigger the issue
2. **Isolate**: Narrow down to the specific location
3. **Hypothesize**: Form theories about the cause
4. **Test**: Add logging or checks to verify theories
5. **Fix**: Implement minimal fix once cause is found
6. **Verify**: Confirm the fix works
7. **Prevent**: Suggest how to prevent similar issues

Ask clarifying questions if the symptom description is vague.
